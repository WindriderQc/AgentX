Methodology & Scoring Guide

×

Quality Scoring (LLM-as-Judge)

We use a capable "Judge" model to evaluate responses from other models. The judge scores each response on a 0-10 scale based on specific criteria tailored to the prompt type:

  • Reasoning: Evaluates accuracy, logical flow, and clarity.
  • Code: Checks for correctness, code clarity, and efficiency.
  • Factual: Verifies accuracy, completeness, and presentation.
  • Math: Checks the final answer, solution method, and steps.
  • Creative: Assesses creativity, coherence, and relevance.

Composite Score Calculation

The Composite Score provides a single balanced metric (0-10) to rank models overall. It is calculated using a weighted formula:

50% Quality Judge's score
30% Latency Response start time
20% Speed Tokens per second
